# Optical Distance Monitor

The IoTsens LoRaWAN® Optical Distance Monitor is equipped with a LIDAR sensor for monitoring filling levels. It becomes an optimal tool in waste management, and can also be used in the detection of liquid levels in tanks, among other applications. The device provides accurate distance measurement with integrated temperature compensation and battery monitoring capabilities.

## Features

- **LIDAR Technology**: Optical distance measurement for accurate level detection
- **Multi-Purpose Applications**: Waste management, liquid level monitoring in tanks, and distance detection
- **Integrated Sensors**: Distance, battery level, and temperature monitoring
- **Compact Design**: 90 x 190 x 80 mm dimensions
- **Robust Construction**: IP65 enclosure rating for outdoor and industrial use
- **Wide Operating Range**: -40°C to 65°C temperature tolerance
- **LoRaWAN Connectivity**: Long-range wireless communication with low power consumption
- **Battery Powered**: Designed for autonomous operation

## Use Cases

- **Waste Management**: Monitor filling levels in waste containers and bins
- **Tank Level Monitoring**: Measure liquid levels in storage tanks and reservoirs
- **Smart Cities**: Optimize collection routes and reduce operational costs
- **Industrial Applications**: Distance and level detection in various industrial environments

## Thinger.io Integration

The IoTsens Optical Distance Monitor integrates seamlessly with Thinger.io through LoRaWAN network servers, enabling real-time monitoring, data visualization, and automated alerts for filling levels and device status.

## Requirements

A LoRaWAN server is required to communicate the IoTsens Optical Distance Monitor into Thinger.io, some options are:

- [The Things Stack](https://www.thethingsindustries.com/stack/)
- [LORIOT](https://loriot.io/)
- [ChirpStack](https://www.chirpstack.io/)

Alongside, the corresponding plugin for the selected LoRaWAN server needs to be installed in your Thinger.io instance.

## Get Started

### Installation

Look for the plugin in the [Thinger.io Plugin Store](https://marketplace.thinger.io/) and install it in your Thinger.io instance. Once the plugin is installed a new Product will be created for this device.

### Configuration

The Product is already preconfigured, check that the auto provision prefix matches the one selected in your LoRaWAN server plugin in Thinger.io, or change it to your desire.

### Usage

Start sending uplinks for autoprovisioning devices and buckets.
This product also provides a predefined dashboard and downlinks.
