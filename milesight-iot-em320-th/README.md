# EM320-TH - Temperature & Humidity Sensor

The EM320-TH is a simple, compact and powerful LoRaWAN®-based device for precise temperature and humidity monitoring. It is equipped with a Sensirion high-precision sensor capable of sensing subtle changes in temperature and humidity, making it ideal for accurate data detection in various harsh environments. The device features an IP67 waterproof enclosure with multi-level structure and sealing ring design, ensuring strong waterproof performance. It uses FDA-approved food-grade material enclosure, which is safe for contact with food or medicines, and is EN12830 certified for cold-chain applications. With ultra-low power consumption and a 5400 mAh built-in replaceable battery, the EM320-TH ensures durable battery life. The magnetic mounting version allows easy attachment to metal surfaces such as freezers, making it particularly suitable for cold chain monitoring, pharmacy environment monitoring, and refrigerator temperature tracking.

## Features

- Sensirion high-precision temperature and humidity sensor
- IP67 waterproof enclosure for harsh environments
- FDA-approved food-grade material enclosure
- EN12830 certified for cold-chain applications
- Ultra-low power consumption with 5400 mAh replaceable battery
- Magnetic mounting option for easy installation
- Data storage capability when network fails
- Operating temperature range: -30°C to 60°C
- Compact dimensions: 58 x 18 x 85 mm
- LoRaWAN® connectivity

## Use Cases

- Cold chain monitoring and data tracking
- Pharmacy and healthcare environment monitoring
- Food storage and refrigerator temperature monitoring
- Warehouse and logistics temperature control
- Laboratory and cleanroom environmental monitoring
- Agriculture and greenhouse monitoring

## Thinger.io Integration

The EM320-TH integrates seamlessly with Thinger.io through LoRaWAN network servers, enabling remote temperature and humidity data monitoring, visualization, and management.

## Requirements

A LoRaWAN server is required to communicate the EM320-TH into Thinger.io, some options are:

- [The Things Stack](https://www.thethingsindustries.com/stack/)
- [LORIOT](https://loriot.io/)
- [ChirpStack](https://www.chirpstack.io/)

Alongside, the corresponding plugin for the selected LoRaWAN server needs to be installed in your Thinger.io instance.

## Get Started

### Installation

Look for the plugin in the [Thinger.io Plugin Store](https://marketplace.thinger.io/) and install it in your Thinger.io instance. Once the plugin is installed a new Product will be created for this device.

### Configuration

The Product is already preconfigured, check that the auto provision prefix matches the one selected in your LoRaWAN server plugin in Thinger.io, or change it to your desire.

### Usage

Start sending uplinks for autoprovisioning devices and buckets.
This product also provides a predefined dashboard and downlinks.
