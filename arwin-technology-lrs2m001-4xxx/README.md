# LRS2M001-4P3P - Power Monitoring Sensor

The Arwin Technology LRS2M001-4P3P is a LoRaWAN-enabled power monitoring sensor designed for AC current metering, electric smart metering, voltage monitoring, and energy management applications. This compact device provides real-time monitoring of electrical parameters, enabling remote power consumption tracking and energy optimization in industrial, commercial, and residential environments.

## Key Features

- **Multi-parameter Monitoring**: Measures voltage, current, energy, and power
- **LoRaWAN Connectivity**: Long-range, low-power wireless communication
- **Compact Design**: 112 x 70 x 39 mm housing
- **Wide Operating Range**: -20°C to 70°C temperature tolerance
- **AC Current Metering**: Real-time current measurement capabilities
- **Energy Management**: Track and optimize power consumption remotely

## Use Cases

- Electric smart metering
- Voltage monitoring and regulation
- Energy consumption analysis
- Power quality assessment
- Remote facility management
- Industrial equipment monitoring
- Building automation systems

## Thinger.io Integration

The LRS2M001-4P3P integrates seamlessly with Thinger.io through LoRaWAN connectivity, enabling real-time data visualization, alerting, and energy management automation.

## Requirements

A LoRaWAN server is required to communicate the Arwin Technology LRS2M001-4P3P into Thinger.io, some options are:

- [The Things Stack](https://www.thethingsindustries.com/stack/)
- [LORIOT](https://loriot.io/)
- [ChirpStack](https://www.chirpstack.io/)

Alongside, the corresponding plugin for the selected LoRaWAN server needs to be installed in your Thinger.io instance.

## Get Started

### Installation

Look for the plugin in the [Thinger.io Plugin Store](https://marketplace.thinger.io/) and install it in your Thinger.io instance. Once the plugin is installed a new Product will be created for this device.

### Configuration

The Product is already preconfigured, check that the auto provision prefix matches the one selected in your LoRaWAN server plugin in Thinger.io, or change it to your desire.

### Usage

Start sending uplinks for autoprovisioning devices and buckets.
This product also provides a predefined dashboard and downlinks.
